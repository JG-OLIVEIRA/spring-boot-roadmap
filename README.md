# SPRING BOOT ROADMAP

<img src="./src/spring-boot-roadmap.png">

1. Java Fundamentals
java-fundamentals-deepdive (Abrangendo de OOP a Pattern Matching)

java-concurrency-multithreading (Tópico específico e complexo)

2. Build Tools & Core
spring-maven-gradle-structure

spring-framework-core-ioc (IoC, DI, Beans e ApplicationContext)

3. Spring Boot Essentials
spring-boot-essentials-config (Autoconfig, Profiles, Actuator)

4. Data Access & Persistence
spring-data-jpa-hibernate (Transações, Fetch types, Queries)

spring-database-migrations (Flyway ou Liquibase)

spring-persistence-databases (PostgreSQL, MySQL, Redis)

5. Building REST APIs
spring-api-rest-standard (Controllers, DTOs, Validation, Pagination)

spring-api-exception-handling (ControllerAdvice e Status Codes)

6. External Communication & Testing
spring-api-clients-webclient (RestTemplate e WebClient)

spring-testing-unit-integration (JUnit 5, Mockito, MockMvc)

spring-testing-testcontainers (Focado em testes de infraestrutura com DB)

7. Security
spring-security-jwt-auth (Filtros, JWT e Roles)

spring-security-oauth2-oidc (Keycloak ou Auth0)

8. Advanced Topics
spring-reactive-programming (WebFlux)

spring-messaging-rabbitmq-kafka

spring-microservices-architecture (GraphQL, SSO)

9. DevOps & Cloud
spring-devops-monitoring (Docker, ELK, Prometheus/Grafana)

spring-cloud-deployment-k8s (AWS/GCP/Azure e Kubernetes)
